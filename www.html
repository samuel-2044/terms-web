World Wide Web (WWW)

The foundation of modern Internet browsing and information sharing.

What is the World Wide Web?

The World Wide Web is a system of interlinked hypertext documents accessed via the Internet. It allows users to browse and share information using web browsers and hyperlinks.

Inventor

The WWW was invented by Sir Tim Berners-Lee in 1989 at CERN and became publicly available in 1991.

Key Components

  • Web Browsers: Access and display web pages
  • Web Servers: Store and deliver content
  • Protocols: HTTP/HTTPS, HTML, CSS, JavaScript

How It Works

A browser sends a request to a web server using a URL. The server responds with a web page, which is then displayed in the browser.

Impact on Society

  • Global communication and instant information access
  • Enabled e-commerce and online services
  • Revolutionized education and entertainment

Evolution of the Web

  • Web 1.0: Static read-only pages
  • Web 2.0: Interactive social web
  • Web 3.0: AI + decentralized systems
  • Web 4.0: Intelligent and predictive web

💡 The WWW is the system that made the Internet usable for everyone.

How the WWW Works

  1. User Action: Type URL or click link in browser
  2. DNS Lookup: Browser queries DNS server for IP address
  3. TCP Connection: Establishes connection via TCP three-way handshake
  4. HTTP Request: Browser sends HTTP request to web server
  5. Server Processing: Server processes request, may query database
  6. HTTP Response: Server sends HTML, CSS, JavaScript files
  7. Browser Rendering: Browser parses HTML, applies CSS, executes JS
  8. Display: User sees the rendered web page
  9. Interactivity: User clicks, types, navigates (AJAX updates)

Key Technologies of the WWW

  • HTML: Content structure and semantics
  • CSS: Visual styling and layout
  • JavaScript: Interactivity and dynamic behavior
  • HTTP/HTTPS: Data transfer protocol
  • URLs: Resource addressing system
  • Web Browsers: Content rendering engines
  • Web Servers: Content storage and delivery

WWW vs Internet

Internet: The global network infrastructure of computers, cables, routers, and protocols

World Wide Web: A service that operates over the Internet, using HTTP to access web pages

  • The Internet is the highway system; the WWW is one type of vehicle on it
  • Internet uses many protocols (SMTP, FTP, SSH, TCP/IP); WWW mainly uses HTTP/HTTPS
  • Internet existed before the WWW (1960s vs 1989)
  • Other Internet services: Email, FTP file transfer, SSH remote access, DNS

Search Engines and the WWW

  • Google: Founded 1998, uses PageRank algorithm (70%+ market share)
  • Crawling: Bots discover and index web pages
  • Indexing: Organize content for fast retrieval
  • Ranking: Determine relevance and quality of pages
  • Over 5 billion web pages indexed across all search engines

The Future of the WWW

  • Semantic Web: Data understandable by machines (Web 3.0)
  • Progressive Web Apps (PWAs): App-like experience in browsers
  • WebAssembly: Near-native performance in web apps
  • Voice and AR interfaces: Beyond traditional screens
  • Decentralized Web: Blockchain-based distributed systems

📊 Over 1.8 billion websites exist, with 250+ million active sites receiving visitors daily.
