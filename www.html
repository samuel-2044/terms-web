World Wide Web (WWW)

The foundation of modern Internet browsing and information sharing.

What is the World Wide Web?

The World Wide Web is a system of interlinked hypertext documents accessed via the Internet. It allows users to browse and share information using web browsers and hyperlinks.

Inventor

The WWW was invented by Sir Tim Berners-Lee in 1989 at CERN and became publicly available in 1991.

Key Components

  • Web Browsers: Access and display web pages
  • Web Servers: Store and deliver content
  • Protocols: HTTP/HTTPS, HTML, CSS, JavaScript

How It Works

A browser sends a request to a web server using a URL. The server responds with a web page, which is then displayed in the browser.

Impact on Society

  • Global communication and instant information access
  • Enabled e-commerce and online services
  • Revolutionized education and entertainment

Evolution of the Web

  • Web 1.0: Static read-only pages
  • Web 2.0: Interactive social web
  • Web 3.0: AI + decentralized systems
  • Web 4.0: Intelligent and predictive web

💡 The WWW is the system that made the Internet usable for everyone.
