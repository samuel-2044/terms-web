Web 1.0

Web 1.0 refers to the first stage of the World Wide Web's evolution, characterized by static web pages and limited user interaction. In this era, websites were primarily informational, with content created by a few authors and consumed by many users. There was minimal user-generated content or interactivity.

Web 1.0 technologies included basic HTML, GIF images, and simple forms. Examples of Web 1.0 sites include early versions of Yahoo!, Netscape, and personal homepages.
