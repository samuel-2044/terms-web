Hypertext Markup Language (HTML)

The standard language used to structure web pages.

What is HTML?

Hypertext Markup Language (HTML) is the standard markup language used to create and structure web pages. It defines elements such as headings, paragraphs, links, images, and multimedia content.

How HTML Works

HTML provides the structure of a webpage. Web browsers read HTML documents and render them into visible web pages for users.

Key Components

  • Elements: Building blocks of HTML (e.g., <p>, <h1>)
  • Tags: Used to define elements
  • Attributes: Provide additional information about elements

HTML with Other Technologies

  • CSS: Adds styling and layout
  • JavaScript: Adds interactivity and logic

HTML5

HTML5 is the latest version of HTML. It introduced semantic tags like <header>, <section>, and <footer>, as well as built-in support for audio, video, and modern web APIs.

💡 HTML is the foundation of every website on the internet.

HTML Document Structure

A basic HTML5 document follows this structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

Semantic HTML Elements

HTML5 introduced semantic elements for better structure and SEO:

  • <header> - Introductory content or navigational links
  • <nav> - Navigation section
  • <main> - Main content of the document
  • <article> - Self-contained composition (blog post, news)
  • <section> - Thematic grouping of content
  • <aside> - Sidebar or tangential content
  • <footer> - Footer information (copyright, links)
  • <figure> - Self-contained media content
  • <figcaption> - Caption for figure element

Essential HTML Attributes

  • id: Unique identifier for an element
  • class: One or more class names for CSS/JavaScript
  • src: Source file for images, scripts, iframes
  • href: Destination for links and references
  • alt: Alternative text for images (accessibility)
  • title: Advisory information on hover
  • style: Inline CSS styling
  • data-*: Custom data attributes for JavaScript

Common HTML Elements

Element Description Example
<p> Paragraph <p>Hello World!</p>
<a> Anchor/Link <a href="page.html">Click here</a>
<img> Image <img src="pic.jpg" alt="picture">
<ul>/<ol> Unordered/Ordered list <ul><li>Item</li></ul>
<div> Division/container <div class="box">Content</div>
<form> Input form <form><input type="text"></form>
<table> Data table <table><tr><td>Cell</td></tr></table>

HTML Validation

Always validate your HTML to ensure compatibility and accessibility:

  • Use W3C Validator: https://validator.w3.org/
  • Test across multiple browsers (Chrome, Firefox, Safari, Edge)
  • Check accessibility with screen readers
  • Ensure mobile responsiveness
  • Follow WCAG 2.1 guidelines for accessibility
