# 🌐 terms-web

A simple educational website that explains key Internet and Web technologies in a structured and beginner-friendly way.

## 📚 About the Project

**terms-web** is an Internet Terminology learning website designed to help students understand fundamental web concepts such as:

- World Wide Web (WWW)
- TCP/IP Protocol Suite
- Web Servers and Browsers
- URLs and DNS
- HTTP and HTTPS
- Intranet and Extranet
- Multitier Architecture
- FTP and HTML
- Evolution of the Web (Web 1.0 → Web 4.0)

Each topic has its own dedicated page with explanations and examples.

---

## 🧠 Web Evolution Section

The project also includes a full breakdown of web evolution:

- **Web 1.0** – Static, read-only websites  
- **Web 2.0** – Interactive, social, user-generated content  
- **Web 3.0** – AI, blockchain, and decentralized web  
- **Web 4.0** – Intelligent, predictive, and symbiotic web systems
